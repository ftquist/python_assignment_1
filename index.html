How prediction works — no server needed

The trained model weights are baked directly into this HTML file. Everything runs in your browser using JavaScript.

Step 1 — Feature Engineering
• car_age = 2024 − production_year
• log_mileage = ln(1 + mileage) — reduces right skew
• log_levy = ln(1 + levy) — same reason
• Binary: is_leather (1/0), is_left_wheel (1/0)
• Label-encoded: fuel type, gear box, drive wheels, category

Step 2 — StandardScaler
Each feature is normalised: (x − mean) / std
This makes all 12 features comparable in one equation.

Step 3 — Linear Regression
price = 18,947.67 + Σ (coeff_i × scaled_feature_i)

Step 4 — Contribution Breakdown
After prediction, the result panel shows how much each feature raised ↑ or lowered ↓ the final price.
